  • Programmable Gain Amplifier

    • A PGA implementation architecture is designed, consisting of two coarse gain adjustment stages (CGA) and one fine gain adjustment stage (FGA) cascaded in a multi-stage structure. The coarse gain adjustment stages are implemented in an open-loop structure, providing a gain range of 0-18dB with a step size of 9dB. The fine gain adjustment stage is implemented in a closed-loop structure with resistor feedback, achieving a gain range of 0-8dB with a step size of 1dB. The coarse gain stages offer high input impedance, low input noise, and reduced system power consumption. The fine gain stage provides gain accuracy and high load capacity.

    Schematic of the CGA

    Non-original but clever open-loop variable gain amplifier. A brilliant design to enhance the transconductance without increasing the power consumption.

    MORE INFORMATION

    Schematic of the FGA

    Wide common-mode input is achieved by using a digital-controlled input stage. This structure consists of a single NMOS input stage differential pair, which saves power. However, it requires a common-mode voltage detection module. The output uses a Class-AB structure with a floating current source embedded in the amplification stage to reduce power consumption. The linear transconductance loop provides static current bias.
